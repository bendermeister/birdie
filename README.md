# birdie
A little self hostable music streaming service. No user accounts, no nothing.
You create songs, by uploading audio files and naming them. After that you can
attach them albums, artists and apply tags to them for easier searching.

# Feature Goals
- [ ] play songs
- [ ] create songs
  - [ ] upload audio file
  - [ ] name the song
  - [ ] tag the song
- [ ] create albums
  - [ ] name the album
  - [ ] attach songs to the album
  - [ ] attach artists to the album
  - [ ] attach tags to the album
- [ ] create artists
- [ ] create tags
- [ ] query by tag
- [ ] query by song title
- [ ] query by artist
- [ ] query by album
